# 🧧 年兽送福

[iFLow 社区](https://vibex.iflow.cn/) 2026 新春创造营参与作品：一款基于 Phaser 3 开发的春节主题 2D 横版跑酷游戏。

## 🎮 游戏简介

扮演可爱的年兽，在春节的氛围中奔跑跳跃，收集福字和红包，躲避鞭炮和灯笼，完成送福使命！

## ✨ 主要功能特色

### 🎯 核心玩法
- **横版跑酷**：自动卷轴前进，速度随进度递增
- **多状态操控**：奔跑、跳跃（支持二段跳）、下蹲躲避
- **自由飞行**：拾取"春"字激活限时飞行模式，自由控制移动
- **无敌冲刺**：消耗能量进入无敌状态，可撞毁障碍物

### 🏮 春节元素
- **福字收集**：铜福(+10分)、银福(+25分)、金福(+50分)
- **红包系统**：收集红包解锁成就奖励
- **年兽主角**：Q版年兽角色，带有奔跑、跳跃、下蹲、受伤动画
- **节日氛围**：飘落花瓣/红包粒子、烟花庆祝特效

### 🌟 关卡设计
- **3大主题场景**：乡村→城市→宫殿，各具特色的视差背景
- **渐进难度**：每关长度、速度上限逐步提升
- **Boss战模式**：第三关包含三阶段Boss挑战（随机弹雨→波浪攻势→极限冲刺）

### 🎮 操作方式
- **键盘**：方向键/WASD 控制移动、跳跃、下蹲
- **触屏**：虚拟摇杆（移动端自动显示）
- **ESC键**：暂停游戏

### 🏆 成就系统
8个可解锁成就：
- 福气满满 - 累计收集100个福字
- 红包收藏家 - 单局收集20个红包
- 爆竹免疫 - 无伤通过第三关
- 快递高手 - 30秒内完成第一关
- 平安乡村/都市 - 第一/二关无伤
- 春风得意 - 累计飞行时间60秒
- 金牌快递员 - 通关所有关卡

### 💾 存档功能
- 自动保存最高分
- 关卡解锁进度
- 成就完成情况
- 统计数据分析

### 🎨 技术亮点
- **视差滚动背景**：4层不同速度的背景营造深度感
- **对象池管理**：障碍物和道具复用，优化性能
- **粒子系统**：收集特效、爆炸效果、烟花庆祝
- **响应式布局**：自适应窗口尺寸变化
- **TypeScript 严格模式**：类型安全的游戏开发

## 🚀 快速开始

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建生产版本
pnpm build

# 类型检查
pnpm typecheck
```

## 🛠️ 技术栈

- **游戏引擎**: Phaser 3.90.0
- **语言**: TypeScript 5.9.3
- **构建工具**: Vite 5.1.6
- **包管理器**: pnpm

## 📁 项目结构

```
src/
  scenes/       # 游戏场景（关卡、菜单、暂停等）
  objects/      # 游戏对象（玩家、障碍物、道具）
  managers/     # 管理器（音频、分数、存档等）
  ui/           # UI组件（HUD、虚拟按键、血条）
  utils/        # 工具函数和常量
  types/        # TypeScript类型定义
```
